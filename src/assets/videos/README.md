# Hero Video Setup

The file `hero-drone.mp4` is currently a sample placeholder.

## How to use a High-Quality Drone Video

1.  **Download a video** from one of the free sources below:
    *   [Pexels: Foggy Mountains Drone Shot (4K)](https://www.pexels.com/video/drone-footage-of-mountains-surrounded-by-fog-855018/)
    *   [Pexels: River in Forest (4K)](https://www.pexels.com/video/drone-footage-of-a-river-in-a-forest-2871896/)
    *   [Pixabay: Island Coastline](https://pixabay.com/videos/island-sea-coast-coastline-ocean-2270/)
    *   [Coverr: Nature Drone Collection](https://coverr.co/s/drone-nature)

2.  **Rename the downloaded file** to `hero-drone.mp4`.

3.  **Replace** the existing file in this directory:
    `src/assets/videos/hero-drone.mp4`

4.  The application will automatically show your new video in the hero background.
